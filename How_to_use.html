Page setup

The first thing to use in your HTML is <page></page> tags. These tags take in values of "class" attributes the format of the page as well as a margin.
For example if I want a page in A4 format and a default margin of 10mm :

<page class ="A4 default-margin"> </page>

For the moment, it is only available pages from A3 to A6 format.
And concerning the margins you have the choice between "no-margin"(0mm), "small-margin"(5mm), "default-margin"(10mm) and "big-margin"(20mm) .
